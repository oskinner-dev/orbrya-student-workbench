### Run Locally

#### Prerequisites
- [.NET 8 SDK](https://dotnet.microsoft.com/download) or higher
- Modern web browser (Chrome 90+, Edge 90+, Firefox 88+)

#### Installation

```bash
# Clone the repository
git clone https://github.com/oskinner-dev/orbrya-student-workbench.git
cd orbrya-student-workbench

# Navigate to project
cd WasmApp

# Build the project
dotnet build

# Run the development server
dotnet run
```

Open your browser to `http://localhost:5000`
